Radio button hanya akan menjadi satu grup pilihan jika memiliki atribut "name" yang sama. Jika tidak diberi atribut name atau nilai name berbeda, maka setiap radio button dapat dipilih secara bersamaan karena dianggap sebagai grup yang berbeda.

Checkbox memungkinkan pengguna memilih lebih dari satu pilihan. Atribut "name" tidak mempengaruhi jumlah pilihan yang bisa dipilih, berbeda dengan radio button. Pada checkbox, atribut name hanya digunakan untuk pengelompokan data saat form dikirim ke server.

Uji Coba 4




Input type="color" digunakan untuk memilih warna melalui color picker yang disediakan browser, sehingga pengguna dapat memilih warna dengan lebih mudah dan visual. Input type="date" digunakan untuk memilih tanggal melalui kalender otomatis dari browser, sehingga meminimalkan kesalahan format penulisan tanggal. Sedangkan input type="button" digunakan untuk menjalankan perintah tertentu, seperti menampilkan pesan "Hello World" ketika tombol diklik. Ketiga tipe input ini menunjukkan bahwa HTML menyediakan berbagai jenis input untuk mempermudah interaksi pengguna.
